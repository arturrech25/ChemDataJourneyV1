# PRD ChemData Journey

## Objetivo
Criar um dashboard offline em HTML, CSS e JavaScript para acompanhar a transição de carreira para Data Science aplicada à Química.

## Estrutura
- Dashboard
- Cursos
- Projetos
- Estatísticas
- Vagas
- Diário
- Configurações

## Tecnologias
- HTML5
- CSS3
- JavaScript
- LocalStorage
- Chart.js (CDN)

## Restrições
- Sem React
- Sem Angular
- Sem Vue
- Funcionar apenas abrindo index.html

## Estrutura de arquivos
```
index.html
style.css
app.js
roadmap.js
storage.js
charts.js
assets/
data/
```

## Dashboard
Cards de progresso, horas estudadas, próxima tarefa, timeline de 24 semanas, gráfico.

## Cursos
Checklist, progresso, certificado e link.

## Projetos
Objetivo, status, tecnologias e GitHub.

## Estatísticas
Gráficos, horas, evolução semanal.

## Vagas
Compatibilidade baseada em Python, Pandas, SQL, Estatística, ML, Git e Power BI.

## Diário
Notas diárias com salvamento automático.

## Configurações
Tema, meta diária, exportação/importação JSON.

## Roadmap
6 meses: Python, ML, SQL, Estatística, Kaggle/GitHub e Projeto Final.

## Design
Inspirado em GitHub, Notion e Linear.
